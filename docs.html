DOCUMENTAÇÃO

Guia completo para utilizar a API do Cyberpunk 2077. Todos os endpoints retornam dados em formato JSON.

Base URL

https://jose-pires-neto.github.io/Cyberpunk-2077-API/docs/api/v1/
GET /characters.json

Retorna a lista completa de personagens do jogo.

Exemplo de Resposta

[
  {
    "id": 1,
    "name": "Adam Smasher",
    "gender": "Male",
    "directory": "adam_smasher",
    "has_images": true,
    "images": [
      "https://...images/characters/sex/male/adam_smasher/adam_smasher_01.png",
      "https://...images/characters/sex/male/adam_smasher/adam_smasher_02.png"
    ],
    "description": "Adam Smasher is a full borg mercenary...",
    "affiliation": "Arasaka",
    "occupation": "Solo",
    "status": "Alive"
  }
]

Campos

Campo Tipo Descrição
id number Identificador único
name string Nome do personagem
gender string Gênero (Male/Female/Unknown)
has_images boolean Se possui imagens disponíveis
images array URLs das imagens
description string Biografia do personagem
affiliation string Gangue/Corporação/Profissão
occupation string (opcional) Profissão
status string (opcional) Vivo/Morto
wiki_url string (opcional) Link para Wiki
GET /gangs.json

Retorna informações detalhadas sobre as gangues de Night City.

Exemplo de Resposta

[
  {
    "id": 1,
    "name": "Maelstrom",
    "directory": "maelstrom",
    "description": "Gangue obcecada por cyberware...",
    "founder": "Boz",
    "leader": "Royce",
    "hq": "Totentanz",
    "territory": "Northside Industrial District",
    "affiliations": ["Scavengers"],
    "wiki_url": "https://cyberpunk.fandom.com/...",
    "images": ["https://.../maelstrom_01.png"]
  }
]

Campos

Campo Tipo Descrição
id integer ID único
name string Nome da gangue
description string Descrição
founder string (opcional) Fundador
leader string (opcional) Líder atual
hq string (opcional) Quartel-general
territory string (opcional) Território
affiliations array (opcional) Afiliações
images array URLs das imagens
GET /districts.json

Retorna os distritos de Night City com subdistritos aninhados.

Exemplo de Resposta

[
  {
    "id": 1,
    "name": "Watson",
    "description": "Distrito decadente no norte...",
    "danger_level": "Moderate",
    "wiki_url": "https://cyberpunk.fandom.com/...",
    "images": ["https://.../watson_01.png"],
    "subdistricts": [
      {
        "name": "Kabuki",
        "description": "Área com mercado de rua...",
        "images": ["https://.../kabuki_01.png"]
      }
    ]
  }
]

Campos do Distrito

Campo Tipo Descrição
id integer ID único
name string Nome do distrito
description string Descrição
danger_level string (opcional) Nível de perigo
images array URLs das imagens
subdistricts array Lista de subdistritos

Campos do Subdistrito

name string Nome do subdistrito
description string (opcional) Descrição
images array URLs das imagens
GET /affiliations.json

Lista de afiliações disponíveis (gangues, corporações, profissões).

Exemplo de Resposta

[
  {
    "id": "maelstrom",
    "name": "Maelstrom",
    "category": "gang",
    "color": "#ff0000"
  },
  {
    "id": "arasaka",
    "name": "Arasaka",
    "category": "corporation",
    "color": "#ff0033"
  }
]

Exemplo de Uso

JavaScript (Fetch)

fetch('https://jose-pires-neto.github.io/Cyberpunk-2077-API/docs/api/v1/characters.json')
  .then(response => response.json())
  .then(characters => {
    // Filtrar apenas personagens COM imagens
    const withImages = characters.filter(c => c.has_images !== false);
    console.log(withImages);
  });

Python

import requests

url = "https://jose-pires-neto.github.io/Cyberpunk-2077-API/docs/api/v1/characters.json"
response = requests.get(url)
characters = response.json()

# Filtrar apenas personagens COM imagens
with_images = [c for c in characters if c.get('has_images', True)]

for char in with_images[:5]:
    print(f"{char['name']} - {char['affiliation']}")

⚡ Rate Limits

Esta API é servida via GitHub Pages e não possui rate limits específicos. No entanto, recomendamos fazer cache local dos dados para melhor performance.
